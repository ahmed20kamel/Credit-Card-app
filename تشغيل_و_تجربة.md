# تشغيل التطبيق وتسجيل الدخول

## 1. تشغيل الباكند (Django)

```bash
cd backend
source venv/bin/activate
python manage.py runserver 0.0.0.0:8000
```

## 2. إنشاء مستخدم تجريبي (مرة واحدة)

```bash
cd backend
source venv/bin/activate
python manage.py create_test_user
```

سيظهر لك:
- **Email:** test@cardvault.local  
- **Password:** TestPass123!

## 3. تشغيل الفرونت (Next.js)

```bash
cd frontend
npm run dev
```

## 4. تسجيل الدخول

1. افتح المتصفح: **http://localhost:8080**
2. استخدم بيانات المستخدم التجريبي:
   - **Email:** `test@cardvault.local`
   - **Password:** `TestPass123!`
3. اضغط Sign in.

## من جهاز آخر على نفس الشبكة

- من جهازك: http://localhost:8080
- من الجوال/كمبيوتر آخر: http://**IP-جهازك**:8080 (مثال: http://192.168.1.53:8080)  
  (اعرف الـ IP من التيرمنال: `ipconfig getifaddr en0` على الماك)

## إذا ظهر خطأ 500

- تأكد أن الباكند شغّال على البورت 8000.
- أعد تشغيل الباكند بعد أي تعديل على الإعدادات.
- تأكد من وجود ملف `backend/.env` (يمكن نسخه من `backend/.env.example`).
